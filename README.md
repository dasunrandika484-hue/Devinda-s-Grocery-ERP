# Devinda's Grocery ERP Solution

This is a comprehensive ERP (Enterprise Resource Planning) system developed using **AppSheet**. It is designed to manage day-to-day operations of a grocery store efficiently.

## Core Features
* **Inventory Management:** Track stock levels in real-time.
* **Sales Tracking:** Record daily sales with automated total calculation.
* **Profit Analysis:** Calculate net profit per item and per sale.
* **Supplier Management:** Maintain a database of suppliers and track outstanding debts.
* **Automated Invoicing:** Generate PDF invoices automatically for every sale.
* **Business Dashboard:** A centralized view to monitor both inventory and sales.

## Tech Stack
* **Platform:** Google AppSheet (No-code)
* **Database:** Google Sheets
* **Automation:** AppSheet Automation for PDF Generation
